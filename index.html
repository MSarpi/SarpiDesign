MasTransporte - Empresa


Se trata sobre Contenido se envios o despachos de paqueteria o bultos a nivel nacional para tiendas ecommerce, empresas y negocios. Todo lo que necesitas para gestionar tus envios y despachos.

Mi Proposito en la empresa era mejora el sistema tanto exterior como interior

Exterior
  • Mejorar el diseño principal de la pagina.
  • Sistema responivo
  • Crear Seccion de "Contactenos"
  • Crear Seccion de "Busca tu pedido"
  • Mejorar Sistema de Ayuda clientes a base de un contenedor en base de datos
  • Mejorar Sistema de Terminos y coniciones a base de un contenedor en base de datos
Interior
  • Facturacion: Me encargue de crear desde cero un mantenedor de documentos tributarios
  • Recadacion y Cobranza: Me encargue desde cero un mantenedor en donde se pagaba los clientes que poseian alguna deuda (en base a la seccion de facturacion)
  • Mantenedores: Me encargue de crear mantenedores en donde se guaraba la informacion de los terminos y condiciones junto con Ayuda de clientes
Lenguajes usados
  • PHP
  • LARAVEL
  • jQuery
  • JavaScript
  • MYSQL
  • HTML
  • CSS
  • BOOTSTRAP
